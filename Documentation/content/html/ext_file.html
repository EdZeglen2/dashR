The "file" Extension

This codebase extends the basic "file" Context of dashR with more optional (non-standard file) functions for your applications.

Included Methods

dashR("file").bytes_to_size (len);
  • This function returned a text formatted file size ( 2.3GB or 1B or 10.2MB)
  • The parameter len is the size of the file in bytes.
Example use of .bytes_to_size 

        var fileSpec = "c:\\mydocs\\myfile.text";
        var file = dashR("file").get(fileSpec);

        console.log( dashR("file").bytes_to_size(file.Size) );
