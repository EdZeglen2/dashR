The "xml" Context

The "xml" Context of dashR is an extension Context that you may optionally include for changing XML into JSON.

Use of this context is rare and thus it is NOT inclued in the extensions.js file. You must include it manually with a script tag.

Included Methods

dashR("xml").json (xml, tab[optional]);
  • This function returns a JS Object of the xml passed in.
  • The parameter tab is an optional tab character for the output.
Example use of .json
JavaScript 

// from a file

var xml = dashR("file").load("c:\\file.xml");
var jsonObj = dashR("xml").json(xml);

// from an URL

$.ajax({
    url : url,
    type: "GET",
    dataType: "xml",
    success: function (xml)) {
        // fnSuccess(xml);
        var data = dashR("xml").json(xml);
    }
});
dashR("xml").parseXml (xml);
  • This function returns a DOM Object of the xml passed in.
Example use of .parseXml
JavaScript 

// from a file

var xml = dashR("file").load("c:\\file.xml");
var dom = dashR("xml").parseXml(xml);
var json = dashR("xml").xml2json(dom);
