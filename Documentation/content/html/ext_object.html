The "object" Context

The "object" Context of dashR is an extension Context that you may optionally include for handling object related functions.

Included Methods

dashR("object").copy (obj);
  • This function returns a copy of an object.
  • The parameter obj is the object to be copied.
  • Why you may need or want to do this is due to data binding.
Example use of .copy 

            // make me a user
            var user = {
                name: "fred",
                address1: "1313 Mockingbird Lane"
            }

            // make me a copy
            var selectedUser = dashR("object").copy(user);
dashR("object").is_empty (obj);
  • This function returns true or false if the matches {}.
  • The parameter obj is the object to text.
Example use of .is_empty 

            // An empty Object
            var user = {}

            // returns true
            dashR("object").is_empty(user);
dashR("object").map_to (srcObj, mapping);
  • This function is for when you have a large object and you want a smaller object with only some of the copied properties of the large one.
  • The parameter srcObj is the object to copied properties and their values from.
  • The parameter mapping is an array of properties names you want copied into the returned object.
Example use of .map_to 
        var map = [
            "name",
            "contentPath",
            "rootPath",
            "savePath",
            "noOfFiles"
        ];               
        
        var masterObject = {
            name: "name",
            contentPath: "contentPath",
            junk1: "junk1",
            junk2: "junk2",
            rootPath: "rootPath",
            savePath: "savePath",
            junk3: "junk3",
            noOfFiles: "noOfFiles",
        }

        var mapped = dashR("object").map_to(masterObject, map);

        // mapped will be
        {
            name: masterObject["name"],
            contentPath: masterObject["contentPath"],
            rootPath: masterObject["rootPath"],
            savePath: masterObject["savePath"],
            noOfFiles: masterObject["noOfFiles"]
        }
