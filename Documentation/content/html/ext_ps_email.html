The "email" Context

The "email" Context of dashR is an extension Context that you may optionally include for sending an email via Windows PowerShell.

Use of this context is rare and thus it is NOT inclued in the extensions.js file. You must include it manually with a script tag.

Included Methods

dashR("xml").power_shell.send (config, message, folder);
  • This function creates a PowerShell .ps1 file in the folder and sends an email with optional HTML and attachments.
  • This is actually creating a text file and running powershell the file name is the message.id which you may override if need be.
  • The location of powershell.exe is defaulted and specified in config and can be overridden.
  • See the unminifed raw code in the files tab for more config and message options not shown here.
  • See below for an example.
Example use of .power_shell.send
JavaScript 

var config = new dashR("email").config(
    "your_smtp_server", 
    "your_smtp_user_name", 
    "your_smtp_pass_word"
);

// override the default port of 587
//config.port = 25;

var message = new dashR("email").message(
    "ezeglen@gmail.com",                // to
    "ezeglen@twc.com",                  // from
    "See Attached",                     // subject
    "This is <br> a test"         // message HTML
);

// optionally turn off HTML in the body
// message.isHTML = false; 

// optionally add attachment(s)
message.attachments.push(this.app.folder + "\\Data\\attachment1.txt");

dashR("email").power_shell.send(config, message, this.app.folder);
