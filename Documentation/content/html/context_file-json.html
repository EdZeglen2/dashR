The "json" Context

The "json" Context of dashR is included for performing the loading and saving of JSON objects (and arrays) to and from local files.

Included Methods

dashR("json").load (fileSpec, defObj[optional]);
  • The fileSpec of the JSON file to load
  • When the file is not found (or is empty) the function returns the defObj
dashR("json").save (fileSpec, objOrArray);
  • The fileSpec to save the data to
  • objOrArray is the object (or array) to JSON.stringify into the file.

Usage Examples 

// save my items to a file for later.
dashR("json").save("c:\\myfile1.json", this.items);

// save the obj to a file for later.
var obj1 = {
    "name": "bob",
    "id" : 0,
    "cars": [
        "porche", "bmw"
    ]
};

dashR("json").save("c:\\myfile1.json", obj1);

// === 

// get me a list of items I saved before.
// if the file does not exist, return and empty array.
this.items = dashR("json").load("c:\\myfile1.json", []);

// load my obj or return an empty one.
var obj1 = dashR("json").load("c:\\myfile1.json", {
    "name": "",
    "id" : 0,
    "cars": []    
});
