The "date" Context

The "date" Context of dashR is an extension Context that you may optionally include for handling date related functions. This is included via in the dashR-DateTime extension.

Included Methods

dashR("date").is_date (obj);
  • The parameter obj is the obj to test.
Example use of .is_date 

            var items = [];

            var stringText = "10-09-2024";

            if (dashR("date").is_date(stringText)) {
                console.log("its a date!");
            } else {
                // its not a date! :-(
            }

            if (dashR("date").is_date(items)) {
                console.log("its a date!");
            } else {
                // its not a date! :-(
            }
dashR("date").format (text, format);
  • The parameter text is the text to make a date from.
  • The parameter format is how to format the date returned.
  • formats include:
    "mm-dd-yyyy"
    "mm/dd/yyyy"
    "mm/dd/yyyy hh:mm:ss ampm"
    "day mon dd yyyy"
Example use of .format 

        var today = new Date();
        console.log(dashR("date").format(today, "mm-dd-yyyy"));
        \\ returns: 10-09-2024
dashR("date").from (jsonStrDate);
  • The parameter jsonStrDate is the MS JSON text date example: /Date(1731506129972)/.
Example use of .from 

        var today = "/Date(1731506129972)/";
        console.log(dashR("date").from(today));
        \\ returns: 10-09-2024
