The "system" Context

The "system" Context of dashR is included for performing OS related tasks like spawning another application.

Included Methods

dashR("system").cmd (fileSpec, cmdargs, windowstyle[optional], waitonreturn[optional]);
  • The fileSpec of the file to open using the command window
  • Any command line paramters
dashR("system").open (urlOrFileSpec);
  • The urlOrFileSpec to spawn using the operating systems default for the type

Usage Examples 

// perform command line operations
var vsCode = "c:\\programfiles\vscode.exe";
dashR("system").cmd (vsCode, "c:\\user\\text.json");

// spawn open other applications
dashR("system").open ("c:\\user\\text.txt");        // spawn Notepad
dashR("system").open ("c:\\user\\word.doc");        // spawn Word
dashR("system").open ("https://www.google.com");    // spawn the default Browser
