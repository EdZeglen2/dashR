The "time" Context

The "time" Context of dashR is an extension Context that you may optionally include for handling time related functions.

Included Methods

dashR("time").format (text, format);
  • The parameter text is the text to make a time from.
  • The parameter format is how to format the time returned.
  • formats include:
    "hh:mm ampm"
Example use of .format 

        var today = new Date();
        console.log(dashR("time").format(today, "hh:mm ampm"));
        \\ returns: 08:09 AM
