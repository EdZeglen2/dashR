The "hta" Context

The "hta" Context of dashR is an extension Context that you may optionally include for functions specifically related to HTAs.

Included Methods

dashR("hta").command_line (cmdLine);
  • This function returns an array of the items passed into the HTA from the command line.
  • The parameter text is the command line from the starting of the .HTA file.
  • Important Note: This removes the first parameter as it is the HTA file's name (which is useless as a parameter).
Example use of .format
HTML 
<HTA:APPLICATION
    ID="objFileLoader"
    APPLICATIONNAME="fileLoader"
    SCROLL="yes"
    SINGLEINSTANCE="no"
    WINDOWSTATE="minimize"
    NAVIGABLE="yes"
    ICON="myIcon.ico"
>
JavaScript 
// parse the command line into an array. // ID Above = "objFileLoader"
var params = jDash("hta").command_line(objFileLoader.commandLine); 
if (params.length > 0) {            
    // ... do stuff here ...
}
