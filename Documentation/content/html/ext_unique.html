The "unique" Context

The "unique" Context of dashR is an extension Context that you may optionally include for generating unique IDs.

Included Methods

dashR("unique").GUID (brackets[optional=true]);
  • The parameter brackets is a boolean value indicating if the function returns the GUID with {} or not.
Example use of .format 

        var id3 = dashR("unique").GUID();
        alert(id3);

        var id4 = dashR("unique").GUID(false);
        alert(id4);
dashR("unique").from
  • The following functions return unique values.
    .math()
    .now()
    .time()
    .text(text)
Example use of .from 

        var id1 = dashR("unique").from.math();
        var id2 = dashR("unique").from.now();
        var id3 = dashR("unique").from.time();
        var id4 = dashR("unique").from.text("This is some text");
        
        console.log(id1);
        console.log(id2);
        console.log(id3);
        console.log(id4);
