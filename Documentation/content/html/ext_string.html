The "string" Extension

This codebase extends the basic "string" Context of dashR with more optional functions for your applications.

Included Methods

dashR("string").clean (text);
  • This function removes ASCII characters greater than ASCI Code 127.
  • The parameter text is the text to clean.
  • This should remove the dreaded BOM character and other potential problem characters.
Example use of .clean 
        // Clean it
        var fileText = dashR("string").clean(fileText);
    
        // Save it
        dashR("file").save("c:\\myfile.text", fileText);
dashR("string-ext").starts_with (text);
  • This function returns true if the text starts with find.
  • The parameter text is the text to test.
  • The parameter find is the text to look for.
Example use of .starts_with 
        var someText = "This is some Text";
        var startsWithIs = dashR("string").starts_with(someText, "is"); 
        var startsWithThis = dashR("string").starts_with(someText, "this");
        
        // startsWithIs = false
        // startsWithThis = true
dashR("string").ends_with (text);
  • This function returns true if the text ends with find.
  • The parameter text is the text to test.
  • The parameter find is the text to look for.
Example use of .ends_with 
        var someText = "This is some Text";
        var endsWithIs = dashR("string").ends_with(someText, "is"); 
        var endsWithText = dashR("string").ends_with(someText, "text");
        
        // endsWithIs = false
        // endsWithText = true
