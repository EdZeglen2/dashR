The "list" Context

The "list" Context of dashR is an extension Context that you may optionally include for handling list related functions.

Included Methods

dashR("list").collection ();
  • This returns a simulated JavaScript collection (using an Object)
  • Properties of the collection are:
    .keys
  • Methods of the collection are:
    .item (key)
    .add (key, value)
    .remove (key)
    .clear ()
    .exists (key)
    .length ()
Example use of .collection 
            var coll2 = new dashR("list").collection();

            coll2.add ("ed", "value1");
            coll2.add ("jed", "value2");
            coll2.add ("fred", "value3");

            // NOTE: this is a function and requires ();
            console.log(coll2.length());    

            if (coll2.exists("jed")) {
                //returns: value2
                console.log(coll2.item("jed").value) 
            }

            coll2.clear();
dashR("list").dictionary ();
  • This returns a Windows Scripting.Dictionary Object
  • Official Microsoft Document: Dictionary Object
  • Properties of the collection are:
    .Keys
    .Count
    .Item
    .CompareMode
  • Methods of the collection are:
    .Item (key)
    .Add (key, value)
    .Remove (key)
    .RemoveAll ()
    .Exists (key)
