The "string" Context

The "string" Context of dashR is included for handling basic string functions.

Included Methods

dashR("string").pad (text, char, length, direction[optional]);
  • The text to be padded
  • The char to use for padding
  • The length pads the text with char until this length
  • The (optional) direction to pad, if not supplied it assumes "left" padding
dashR("string").left (str, n);
  • The str is the text to be read
  • The n is how many characters to read
  • [Included for Legacy]
dashR("string").right (str, n);
  • The str is the text to be read
  • The n is how many characters to read
  • [Included for Legacy]
dashR("string").proper_case (text);
  • The text is the text to fix
  • [This is NOT fool-proof and can fail with some text]

Usage Examples 

    // .pad =================================================

    //format date.
    var today = new Date();
    var month = today.getMonth()-1;
    var day = today.getDate();
    var year = today.getFullYear();

    var formatted = dashR("string").pad(month, "0", 2) 
        + "-" + dashR("string").pad(day, "0", 2) 
        + "-" + year;

    console.log(formatted);
    // should return MM-DD-YYYY

    // .left ================================================
    var month = dashR("string").left("10-10-2024", 2);
    // month = 10

    // .right ================================================
    var year = dashR("string").right("10-10-2024", 4);
    // year = 2024

    // .proper_case ==========================================
    var title = dashR("string").proper_case("THIS IS A TITLE");
    // title = This is a Title;
