The "app" Context

The "app" Context of dashR is included primarily so you don't have to create it and for extending your application with your own functions. It contains no properties or methods of its own.

Example of a common use of the "app" context is to immedately extend it with your functions. 


// add a "email" method to the "app" context 
dashR("app")["email"] = function () {
    return {
        send: function (to, messageBody) {
            ...
        }
    }
}

// calling your function
dashR("app").email.send("someone@nowhere.com", "hellow from someone to someone.");
