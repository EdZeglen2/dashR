The "math" Context

The "math" Context of dashR is an extension Context that you may optionally include for handling math related functions.

Included Methods

dashR("math").random_integer (min, max);
  • This function returns a random integer between min and max
  • The parameter min is the lowest integer.
  • The parameter max is the highest integer.
Example use of .format 
        var anInt = dashR("math").random_integer(1, 10);
        console.log(anInt);

        // returns: a random number from 1 to 10
