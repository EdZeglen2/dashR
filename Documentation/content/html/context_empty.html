The empty() Context

The empty() Context of dashR is used for retrieving dashR's intenral objects. The format is dashR().[property or method name]

Included Methods

dashR().versionInfo();
  • Returns an object with the following properties:
    .name
    .date
    .major
    .minor
    .patch
  • Returns an object with the following methods:
    .number()
Example use of .versionInfo 
    var myApp = {
        number: dashR().versionInfo().number();     // dashRs Major.Minor.Patch string (1.0.0)
        ...
    }
dashR().addContext (newContext);
  • The newContext is an object with a name and js { "name": name, "js": js } property.
Example use of .addContext 
    // add a "crypto" context to dashR
    dashR().addContext({
        name: "crypto",
        js: {
            keyLength: 128,
            encrypt: function (text, key) {
                return ...
            },
            decrypt: function (text, hash) {
                return ...
            },
            generateKey: function (password, salt) {
                return ...
            }
        }
    });

    // invoke the generateKey method
    var myKey = dashR("crypto").generateKey("12345", "OctoberWinds");
dashR().newApp (config[optional]);
  • The optional config parameter is an object of propteries.
  • Creates an AppObject
  • Any elements using the unique tag of <dashR></dashR>, will be loaded based on their context and type attributes.
Example use of .newApp
HTML 

        <div id="dialogs"></div>

        <dashR 
            id="selector1" class="icon-selector" 
            name="selector1" context="icons" type="selector"
            > (Loading...) 
        </dashR>
JavaScript 
        // Basic usage
        var myApp = dashR().newApp();
        myApp.load();

        // Advanced with dashR Dialogs support and dashR Packages
        var myApp = dashR().newApp({
            dlgs: {
                folder: "content\\html",
                dialogs: [
                    { name: "dlg-account" },
                    { name: "dlg-sign-on" },
                    { name: "dlg-email" }
                ]
            },
            pkgs: {
                folder: "content\\packages",
                packages: [
                    { name: "icon-selector" },
                    { 
                        name: "file-selector", 
                        css: true, 
                        js: true, 
                        template: false, 
                        folder: "content\\junk", 
                        type: "" 
                    }
                ]                
            }
        });

        myApp.load();
dashR()..el (elName);
  • Returns a reference to a dashR element.
Example use of the <dashR> tags and the .el method 
        <dashR id="myThing1" name="myThing_Name1" context="icons" type="selector"></dashR>
    
        var selector1 = dashR().el("myThing_Name1");
        selector1.init();
        selector1.bind();
